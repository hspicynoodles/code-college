Now in Github there is a tool called "Git". This tool allows us to track changes in our code like when you save your video game after you left off or when you hit the flag when playing mario bros(I love Mario bros). So yeah its just a checkpoint for us when we save/commit our code. Here are some useful commands that we should know when using Git:

  • \git init: Starts a new Git project.
  • \git add: Adds files.
  • \git commit: Commits the staged changes to the repository.
  • \git status: Shows what's changed.
  • \git push: Pushes the local changes to the remote repository.
  • \git pull: Fetches and merges changes from the remote repository.

Now theres are all commands that can be used in your terminal. The temrminal is a command line interface that allows you to interact with your computer using text commands. You can open the terminal in VS Code by going to the top menu and clicking on "Terminal" then "New Terminal" or by clicking ctrl + ` (the backtick key). Now with this knowledge lets go ahead and create a new repository on Github and clone it to your local machine(VS Code) to get the crown!

Okay now that we know what Github is and how to use it lets go ahead and create a new repository and clone it to our local machine. You can always follow my video tutorial for a step-by-step guide!

  • Step 1: Got to github and click "new repository"
  • Step 2: Name your repository "hello-world". You can choose to create a desrcription/public or private/ and a README file. Then click create repository.
  • Step 3: Create a branch. A branch in github is pretty much like a seperate version of your code where you can make changes without messing up the main code so for now we are going to click main.
  • Step 4: Clone the repository to your local machine. You can do this by clicking the "Code" button and copying the URL. Then, in your terminal, navigate to the directory where you want to clone the repository and run the command "git clone [URL]".
  • Step 5: Open the cloned repository in VS Code. You can do this by going to the File menu and selecting "Open Folder..." and then selecting the folder where you cloned the repository.
